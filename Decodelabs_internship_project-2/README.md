# E-Commerce Exploratory Data Analysis (EDA)

## Project Overview
Performed Exploratory Data Analysis on an E-commerce dataset containing 1,200+ order records to discover sales trends, customer behavior, and business insights.

## Objective
Analyze transactional data and generate meaningful insights using data visualization and statistical analysis techniques.

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Steps

###  Data Exploration
- Dataset shape and structure
- Missing values analysis
- Statistical summary

###  Data Cleaning
- Handling missing values
- Duplicate checking
- Date formatting

###  Univariate Analysis
- Product distribution
- Payment methods analysis
- Order status visualization

###  Bivariate Analysis
- Quantity vs Total Price
- Product sales comparison

###  Time Series Analysis
- Monthly sales trends
- Orders over time

## 📈 Key Insights

- Delivered orders represented the highest percentage of orders.
- Credit Card payments were the most used payment method.
- Some products generated significantly higher revenue.
- Sales trends varied across different months.
