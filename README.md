## Overview

This repository contains **all three projects** completed during my internship at **DecodeLabs**. Each project focuses on a different aspect of data analytics, demonstrating practical skills in Excel, Python, and SQL.

##  Projects Structure

| Project | Topic | Tools | Description |
| :--- | :--- | :--- | :--- |
| **Decode_Project_1** | Data cleaning with Python | Pythib, Pandas, Jupyter Notebook | Data cleaning, and sensitivity analysis |
| **Decode_Project_2** | EDA with Python | Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook | Exploratory Data Analysis on e-commerce dataset with visualizations |
| **Decode_Project_3** | SQL Data Analysis | SQL Server, T-SQL, SSMS | Writing optimized queries for filtering, aggregation, and business insights |

##  Project Highlights

### 1️⃣ Decode_Project_1 – Excel Data Analysis
- Cleaned and prepared raw data
- Created interactive pivot tables and dashboards
- Performed sensitivity analysis for business scenarios

### 2️⃣ Decode_Project_2 – Python EDA
- Analyzed 1,200+ e-commerce orders
- Generated visualizations (heatmaps, bar charts, trend lines)
- Extracted insights on sales trends and customer behavior

### 3️⃣ Decode_Project_3 – SQL Data Analysis
- Wrote SELECT, WHERE, GROUP BY, HAVING, ORDER BY queries
- Calculated aggregations: SUM, AVG, MIN, MAX, COUNT
- Identified high-value customers and referral channel performance
- Total revenue analyzed: $1.26M | Average order: $1,054
